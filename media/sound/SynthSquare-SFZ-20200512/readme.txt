Synth Lead Square
-----------------

Version 2020-05-12

Designed to perform as the General MIDI instrument #81: Synth Lead 1 (square).
It has been created by Roberto <roberto@zenvoid.org> with ZynAddSubFX /
Yoshimi software synthesizers.

This sound bank, in several formats and qualities, is available on the
FreePats project web pages: http://freepats.zenvoid.org/
